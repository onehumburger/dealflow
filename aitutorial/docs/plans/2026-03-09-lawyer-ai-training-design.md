# 律师事务所 AI 工具培训 — 设计文档

**日期**: 2026-03-09
**状态**: 已确认

## 项目概述

为小型律师事务所（5-20人）制作2小时AI工具培训。目标：前半段认知升级（理解Chat vs Agent、工具生态），后半段实操演示+互动讨论。

## 受众画像

- 小型律所，合伙人+律师助理，5-20人
- 目前主要使用豆包做简单Chat
- 不了解Agent、Skill、Plugin等概念
- 互动式培训，非讲座

## 核心目标

1. 认知升级：AI不只是聊天机器人，理解Agent/Skill/Plugin的存在和潜力
2. 实操入门：通过live demo展示具体工作场景，让律师知道"我的XX工作可以用YY工具"

## 语言规范

- 中文为主
- 术语用英文原文，附简短中文解释
  - Agent（智能代理）
  - Skill（预设技能模板）
  - Plugin（插件）
  - MCP（Model Context Protocol，模型上下文协议）
  - NotebookLM、OpenClaw 等产品名保持英文

## 课件技术方案

- **框架**: reveal.js，单HTML文件
- **视觉风格**: 深色背景（深蓝/深灰），大字少字，关键术语亮色高亮
- **调性**: "懂技术的朋友在给你讲干货"
- **演示页**: 只放标题 + 演示提示，不嵌入录屏/GIF，全部现场live demo
- **输出**: 浏览器打开即用，支持PDF导出分享

## 培训结构（约120分钟）

### 第一部分：认知升级（50分钟）

#### 1. AI现在能做什么了（10分钟）— 讲解
- 从GPT-3到现在：2年发生了什么
- 上下文窗口的飞跃（4K → 200K tokens，能读整本合同/案卷）
- 多模态能力：读图片、PDF扫描件、表格
- 推理能力的突破：不只是"背答案"，而是能"想问题"

#### 2. Chat vs Agent（15分钟）— 讲解 + live demo
- Chat模式：一问一答，像微信聊天（豆包、ChatGPT网页版）
- Agent模式：AI自主规划、调用工具、多步骤完成任务
- 类比：Chat像实习生等你指令，Agent像资深助理自己安排工作
- Live demo：同一个任务，Chat vs Agent的效率对比

#### 3. 工具生态：Plugin / Skill / MCP（10分钟）— 讲解
- Plugin（插件）：给AI接上专业能力（法律数据库、判例检索）
- Skill（预设技能模板）：专家级提示词 + 预设工作流，不需要自己写复杂prompt
- MCP（模型上下文协议）：让AI连接本地文件、数据库、内部系统
- 类比：手机本身不强，装上APP才强大

#### 4. 法律AI工具版图（15分钟）— 讲解 + live demo
- 国内：豆包、Kimi（长文档）、通义千问
- 国际：Claude（长上下文+推理强）、GPT-4、Gemini
- 法律专用：OpenClaw、Claude Legal Plugin、Harvey AI、CoCounsel
- NotebookLM：上传文档 → 自动生成摘要/问答/播客式讲解，免费、门槛低，适合入门
- 各工具定位和擅长场景对比

### 第二部分：实操+互动（60分钟）

#### 5. 律师工作场景实战（30分钟）— live demo
- 合同审查：上传合同 → AI标注风险条款 → 生成审查意见
- 法律研究：输入案情 → AI检索相关判例+法规 → 生成研究备忘录
- 文书起草：给定要点 → AI生成起诉状/答辩状/法律意见书初稿
- 尽职调查：批量文档分析 → 提取关键信息 → 生成报告摘要
- 客户沟通：将专业法律分析 → 转化为客户能理解的邮件/备忘录

#### 6. 提示词技巧 + Skill（10分钟）— 讲解 + 对比演示
- 基础提示词技巧：角色设定、结构化指令、分步拆解
- Skill的价值：同一个任务，裸prompt vs 用Skill的效果对比
- 核心信息：你不需要成为prompt专家，选对工具/Skill就行
- 常见误区：过于模糊 / 不给上下文 / 不校验输出

#### 7. 互动：你的工作场景匹配（20分钟）— 讨论
- 每人说一个自己工作中最耗时的任务
- 现场帮他们匹配最适合的AI工具和方案
- 答疑解惑

### 闭场（10分钟）

#### 8. 数据隐私（5分钟）— 讨论
- 一页幻灯片列出要点
- 抛出问题让律师自己讨论："你们觉得客户数据上传AI有什么合规考量？"
- AI幻觉：输出必须人工校验，尤其是判例引用

#### 9. 下一步行动建议（5分钟）— 讲解
- 推荐工具清单 + 注册指引
- 建议从哪个场景开始尝试
- 学习资源推荐

## 交付物

1. **reveal.js 课件** — `slides.html`，单文件，浏览器打开
2. **培训讲稿** — `speaker-notes.md`，每页幻灯片的讲解要点和demo操作步骤
3. **资源清单** — 课件最后一页，包含推荐工具链接和注册指引
